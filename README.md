# Algebra App - Python
## Graph Basic Algebra Functions
`Built with customtkinter, matplotlib, and Python math library`

#### To Run
```
Clone repo
pip install -r requirements.txt
python main.py
```

##### Graph following algebraic functions
- Linear
- Quadratic
- Cubic
- Expnential
- Logarithmic
